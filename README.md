Desarrollado en php 7.1, realizado nativamente con componentes de front con bootstrap 3 y con palntilla de adminlte 2.

######  caracteristicas
>- Usuarios
>- Proveedores
>- Ventas
>- Productos
>- Preventas
>- Configuraciones
>- Clientes
>- Categorias

generado para adminstrar puntos de venta, abierto para contribuciones.
